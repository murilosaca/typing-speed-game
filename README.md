# typing-speed-game
A simple browser-based typing challenge. Type the given phrase as fast and accurately as possible before the timer runs out.

Features


60-second countdown timer
Words-per-minute (WPM) calculation
Accuracy percentage
Score tracking
Restart button


Tech Stack


HTML
CSS
Vanilla JavaScript (no dependencies)


How to Run

Just open index.html in any browser — no build step or server required.
